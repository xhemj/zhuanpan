# 班级小转盘

一个为学校班级电脑制作的小转盘工具。

## 技术栈

- [x] Vue
- [x] Element Plus
- [x] Tailwind CSS
- [x] Pinia

## 功能

- [x] 转盘抽奖
- [x] 批量导入名单
- [x] 历史中奖名单
- [x] 编辑奖项、隐藏奖项
